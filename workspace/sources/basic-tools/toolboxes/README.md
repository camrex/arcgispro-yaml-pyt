# Toolboxes Directory

This directory contains **Python Toolbox (`.pyt`) definitions** organized by domain/purpose.

## Structure

Each toolbox folder contains:
- `<name>.pyt` - The Python Toolbox file (opened in ArcGIS Pro)
- `toolbox.yml` - Configuration defining which tools to include
- `*.pyt.xml` - Metadata files (auto-generated by ArcGIS Pro)

## Multiple Toolboxes

This demonstrates the framework's support for **multiple toolboxes** with different tool selections:

### `spatial_analysis/`
Spatial analysis tools for geographic operations:
- Buffer Analysis
- Clip Features

### `utilities/`
Utility tools for toolbox management:
- Load Tool Metadata

## Many-to-Many Relationship

The same tool can appear in multiple toolboxes by simply referencing it in each `toolbox.yml`. Tools are implemented once in `src/tools/` and reused across toolboxes.
